# Flying B Industries Updated Website

This package updates Flying B Industries from a golf-ball-only landing page into a broader custom production website.

## Included Pages

- `index.html` — main homepage
- `uv-printing.html` — custom UV printing service page
- `laser-engraving.html` — laser etching/engraving service page
- `waterproof-decals.html` — waterproof UV decals service page
- `3d-printing.html` — custom 3D printing service page
- `custom-golf-balls.html` — keeps golf balls as a featured specialty
- `quote.html` — static quote request page that opens an email template
- `policies.html` — policies and expectations
- `assets/site.css` — shared styling
- `assets/site.js` — menu, QR banner, and quote email helper
- `robots.txt` and `sitemap.xml`

## Upload Notes

Upload all files to the public root of `flyingbindustries.com`.

This site expects your existing logo file to be available as:

`officialFlyingBLogo.png`

The site includes your existing Google Analytics and Microsoft Clarity snippets from the current page.

## Main positioning

**Your idea. Your item. Made custom.**

The new site supports UV printing, laser engraving, waterproof decals, 3D printed items, custom golf balls, tumblers, leather hat patches, and custom one-off projects.
